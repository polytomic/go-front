# Golang Front API

This is an implementation in Go for [Front API](https://dev.frontapp.com/)

- [x] Token Identity
- [x] Teams
- [x] Teammates
- [x] Inboxes
- [x] Channels
- [x] Conversations
- [ ] Contacts

- [ ] Webhooks
- [ ] Analytics
- [ ] Events
- [ ] Contact groups

- [ ] Tags
- [ ] Rules
- [ ] Exports
